◆ FSM_init()

void FSM_init ( )

Initialize the FSM.

Set FSM_current_state to st_init, q_direction_space to DIRN_STOP, motor direction up and remove all orders.

Postcondition
The elevator will immediately observe if already at a floor, thus triggering further events. Therefore safe to set motor direction regardless of elevator positioning.
